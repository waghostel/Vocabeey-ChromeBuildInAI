# Product Overview

## Language Learning Chrome Extension

A Chrome Extension (Manifest V3) that transforms web articles into interactive language learning experiences using Chrome's built-in AI APIs.

### Core Purpose

- Extract and process web articles for language learning
- Provide vocabulary highlighting and translation
- Create interactive learning cards with TTS support
- Enable sentence-level learning with contextual translations

### Key Features

- **Clean Reading Mode**: Article extraction and processing with card-based UI
- **Dual Highlighting System**: Vocabulary mode and sentence mode
- **AI-Powered Processing**: Chrome Built-in AI APIs with Gemini fallback
- **Progressive Learning**: Difficulty-based content adaptation
- **Offline Capability**: Local storage and caching system

### Target Users

Language learners who want to practice with real-world content from web articles.

### Technical Approach

- Full-page takeover in new tab for learning interface
- Chrome Built-in AI APIs for processing (Summarizer, Translator, Rewriter, Language Detector)
- Fallback to Gemini API for extended functionality
- Local-first data storage with privacy focus
