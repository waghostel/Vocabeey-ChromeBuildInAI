Getting Started

Requirements

  • Chrome Browser: Version 140 or higher
  • Storage: 22GB for AI models (one-time download)
  • RAM: 4GB minimum (8GB recommended)
  • Platform: Windows 10+, macOS 13+, Linux, or ChromeOS

Quick Setup

Step 1

Enable Chrome AI

  1. Open Chrome and go to: chrome://flags/#optimization-guide-on-device-model
  2. Set "Optimization Guide On Device Model" to Enabled
  3. Restart Chrome
  4. AI models download automatically in the background
Step 2

Install Vocabeey

From Chrome Web Store (Coming Soon):

  1. Visit the Chrome Web Store
  2. Click "Add to Chrome"
  3. Start learning!

From Source (For Early Adopters):

  1. Download or clone the repository
  2. Go to chrome://extensions/
  3. Enable "Developer mode"
  4. Click "Load unpacked" and select the extension directory
Step 3

Start Learning

  1. Navigate to any article in your target language
  2. Click the Vocabeey icon in your toolbar
  3. Follow the interactive tutorial (first-time users)
üí° First-Time Experience: When you first open Vocabeey, an interactive tutorial will guide you through all the features. You can replay it anytime from the menu!

‚ùì Frequently Asked Questions

Why doesn't the extension icon work on some pages?

+

Some sites block extensions for security reasons (e.g., Chrome Web Store, banking sites). Try using Vocabeey on article and blog sites where it works best.

The language detection is wrong. How do I fix it?

+

Right-click on the article card and select the correct language from the menu. All your flashcards will update instantly.

My highlights don't appear when I select text

+

Make sure you're in Vocabulary mode (press 1) or Sentence mode (press 2). If you're in None mode (3), highlighting is disabled.

Audio pronunciation doesn't play

+

Check that Chrome AI is enabled in chrome://flags. Make sure "Optimization Guide On Device Model" is set to Enabled and Chrome has been restarted.

The article looks messy with extra content

+

Use Edit Mode to clean up the article before learning. Click on unwanted paragraphs and remove them to keep your learning materials focused.

Can I see all my vocabulary from all article parts?

+

Yes! Switch to Vocabulary mode (press V) to see all vocabulary items from the entire article, not just the current part.

Is my learning data private?

+

Yes! All your learning data is stored locally on your device. Nothing is sent to external servers (except optional Gemini API fallback if you configure it). You have complete control over your data.

Does Vocabeey work offline?

+

Yes! After the initial AI model download (22GB), Vocabeey works completely offline. You can continue learning even without an internet connection.
