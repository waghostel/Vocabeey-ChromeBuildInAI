# User Guide

## Getting Started with Language Learning Reader

Transform any web article into an interactive language learning experience with Chrome's built-in AI.

## Installation

### System Requirements

- **Chrome Browser**: Version 140 or later
- **Operating System**: Windows 10+, macOS 13+, Linux, or ChromeOS
- **Hardware**: 4GB RAM, 22GB storage space, 4GB VRAM (recommended)
- **Internet**: Required for initial setup and Gemini API (optional)

### Install the Extension

1. Visit the Chrome Web Store
2. Search for "Language Learning Reader"
3. Click "Add to Chrome"
4. Confirm installation by clicking "Add extension"

### First-Time Setup

After installation, the setup wizard will guide you through:

1. **Welcome Screen**: Overview of features and capabilities
2. **Language Selection**: Choose your native language for translations
3. **Difficulty Level**: Set your learning level (1-10 scale)
4. **API Configuration**: Optionally add Gemini API key for extended features
5. **Tutorial**: Interactive walkthrough with a sample article

## Basic Usage

### Processing an Article

1. **Navigate to an Article**: Visit any webpage with article content
2. **Click Extension Icon**: Click the Language Learning Reader icon in your toolbar
3. **Wait for Processing**: The extension will extract and process the content
4. **Learning Interface Opens**: A new tab opens with the processed article

## Troubleshooting

### Common Issues

#### Extension Not Working

**Problem**: Extension icon is grayed out or not responding
**Solutions**:

- Ensure you're on a webpage with article content
- Check Chrome version (needs 140+)
- Refresh the page and try again
- Check if Chrome AI is available in your region

#### Content Not Extracting

**Problem**: "No content found" or extraction fails
**Solutions**:

- Try a different article or website
- Check if the page has paywall or login requirements
- Ensure JavaScript is enabled
- Try adding Jina Reader API key for better extraction

#### AI Processing Fails

**Problem**: "AI processing unavailable" error
**Solutions**:

- Check Chrome version and hardware requirements
- Add Gemini API key as fallback
- Try with a shorter article
- Check internet connection

#### Translations Not Working

**Problem**: Vocabulary cards show "Translation failed"
**Solutions**:

- Verify language pair is supported
- Check if Gemini API key is configured correctly
- Try with more common vocabulary words
- Ensure stable internet connection

### Performance Issues

#### Slow Processing

**Problem**: Article processing takes too long
**Solutions**:

- Try shorter articles first
- Check available system memory
- Close other browser tabs
- Consider using Gemini API for faster processing

#### High Memory Usage

**Problem**: Browser becomes slow or unresponsive
**Solutions**:

- Clear old articles from storage
- Reduce number of open learning interface tabs
- Restart browser to free memory
- Check system requirements

### Error Messages

#### "Chrome AI Not Available"

- **Cause**: Browser doesn't support built-in AI
- **Solution**: Update Chrome to version 140+ or add Gemini API key

#### "Storage Quota Exceeded"

- **Cause**: Too much vocabulary/article data stored
- **Solution**: Export data and clear old articles

#### "Network Error"

- **Cause**: Internet connection issues
- **Solution**: Check connection and try again

#### "Invalid API Key"

- **Cause**: Gemini API key is incorrect or expired
- **Solution**: Verify API key in settings

## Frequently Asked Questions

### General Questions

**Q: Is the extension free to use?**
A: Yes, the core features using Chrome's built-in AI are completely free. Gemini API usage requires your own API key and may have associated costs.

**Q: What languages are supported?**
A: Chrome's built-in AI supports major languages including English, Spanish, French, German, Italian, Portuguese, Japanese, and Chinese. Gemini API extends support to many more languages.

**Q: Does it work offline?**
A: Previously processed articles work offline. New article processing requires internet connection for AI services.

**Q: Can I use it on mobile?**
A: Currently designed for desktop Chrome. Mobile support may be added in future versions.

### Technical Questions

**Q: Why do I need Chrome 140+?**
A: Chrome's built-in AI APIs were introduced in version 140. Earlier versions don't have the necessary AI capabilities.

**Q: How much storage does it use?**
A: Varies by usage, but typically 10-50MB for vocabulary and processed articles. The extension monitors usage and offers cleanup when needed.

**Q: Can I sync data across devices?**
A: Currently, data is stored locally per device. Export/import functionality allows manual sync.

**Q: Is my data secure?**
A: Yes, all data stays on your device. No personal information is sent to external servers except when using optional Gemini API with your own key.

### Learning Questions

**Q: What difficulty level should I choose?**
A: Start with level 5 (intermediate) and adjust based on comfort. Beginners might prefer 3-4, advanced learners 7-8.

**Q: How many words should I highlight per article?**
A: Start with 5-10 new words per article. Focus on quality over quantity for better retention.

**Q: Should I read the original or adapted text?**
A: Both! Compare them to understand how language complexity affects meaning while preserving facts.

**Q: How often should I review vocabulary?**
A: Regular review is key. Export your vocabulary to external spaced repetition tools for optimal retention.
