# Usage Examples - Context Menu in None Mode

## Example 1: Adding Vocabulary from None Mode

### Scenario

You're reading an article and come across an unfamiliar word.

### Steps

1. Make sure you're in **None mode** (click üö´ None button or press `0`)
2. Select the word "serendipity"
3. Context menu appears with options
4. Click **"Add as Vocabulary"**
5. Word is highlighted and translated
6. Vocabulary card appears in the sidebar

### Result

- Word "serendipity" is now highlighted in yellow
- Translation appears in your target language
- Hover over the word to see translation popup
- Word appears in vocabulary cards section
- Can review in Vocabulary learning mode

## Example 3: Pronouncing Text Before Adding

### Scenario

You want to hear how a word sounds before adding it.

### Steps

1. In **None mode**, select the word "entrepreneur"
2. Context menu appears
3. Click **"Pronounce"**
4. Listen to the pronunciation
5. If you want to add it, select again and click **"Add as Vocabulary"**

### Result

- Heard the correct pronunciation
- Made an informed decision about adding it
- Word is now in your vocabulary list

---

## Example 4: Trying to Add Too Many Words

### Scenario

You accidentally select too many words for vocabulary.

### Steps

1. In **None mode**, select "the quick brown fox jumps" (5 words)
2. Context menu appears
3. Click **"Add as Vocabulary"**
4. Error tooltip appears: "Please select 1-3 words for vocabulary"

### Result

- Selection is not added
- Clear feedback about the limitation
- Can select fewer words and try again

## Tips for Best Results

### Vocabulary Selection

- ‚úÖ Select 1-3 words
- ‚úÖ Select complete words (not partial)
- ‚úÖ Include context for better translation
- ‚ùå Don't select punctuation only
- ‚ùå Don't select more than 3 words

### Sentence Selection

- ‚úÖ Select complete sentences
- ‚úÖ Include ending punctuation
- ‚úÖ Minimum 10 characters
- ‚ùå Don't select sentence fragments
- ‚ùå Don't select just a few words

### Mode Selection

- **None mode**: For careful, intentional learning
- **Vocabulary mode**: For quick vocabulary collection
- **Sentence mode**: For sentence pattern learning
- Switch modes freely based on your needs

### Context Menu Usage

- Right-click works on desktop
- Long-press works on mobile
- Menu auto-positions to avoid edges
- Click outside menu to close
- Actions are immediate (no confirmation needed)
